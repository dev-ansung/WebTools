# WebTools

A collection of useful static HTML tools hosted on GitHub Pages.

## Tools

- [Image to Base64 converter](image2base.html) — Upload or paste an image, set a target KB size, and get a scaled Base64 data URL output.
- [Base64 to Image converter](base2image.html) — Paste a Base64 data URL and download the resulting image file.
- [Company Logo Finder](logo.html) — Search for a company name and get its logo image URL.
